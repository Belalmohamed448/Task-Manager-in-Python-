# Task Manager

## What it Does

This is a simple Python console application that helps users manage their daily tasks. Users can add tasks, view all tasks, mark tasks as completed, and delete tasks. Each task includes a title, deadline, priority, and creation date.

## How to Run It

1. Make sure Python 3 is installed.
2. Clone this repository:
   ```bash
   git clone https://github.com/Belalmohamed448/Task-Manager-in-Python-.git
   ```
3. Open the project folder.
4. Run the program:
   ```bash
   python task_manager.py
   ```
5. Follow the menu to manage your tasks.
